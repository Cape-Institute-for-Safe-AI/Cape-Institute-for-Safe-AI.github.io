# Fonts

- `Axion.otf` — display typeface used for headings (`--font-display`). Loaded via
  `@font-face` in `assets/css/style.css`. OTF works in all modern evergreen
  browsers; if broader legacy support is ever needed, convert to `.woff2` with
  a tool like [Transfonter](https://transfonter.org/) or `fonttools` and add
  the extra `src` entries.
- Body copy uses **Lato** and the monospace/label type uses **JetBrains Mono**,
  both loaded from Google Fonts in `index.html`.
